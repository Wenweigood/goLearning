# 整型
有符号整型：int8、int16、int32、int64

无符号整型：uint8、unit16、uint32、uint64

<span style="color:grey;font-size:small">与平台有关的整型：int、uint（32/64）</span>

运算符优先级顺序递减排列：
```
*      /      %      <<     >>     &     &^
+      -      |      ^      ==     !=   <      <=       >      >=    &&    ||
```
在Go语言中，%取模运算符的符号和被取模数的符号总是一致的，因此-5%3和-5%-3结果都是-2
